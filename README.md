## Установка и запуск 
Клонируйте репозиторий и войдите в корневую папку проекта 

    git clone git@github.com:alilramazanov/car_shop.git

Войдите в корневую директорию проекта 

    cd car_shop 

Запустите makefile командой make в корнивой директории 

    make
